Ice eye是一个模仿谷歌Kubernetes优点的Zeroc Ice Eye 管理系统，它具有如下的目标和特性：
 - 1.图形化一键安装部署多节点的Zeroc Ice集群，提供默认优化的集群安装参数，实现最佳部署。
 - 2.支持图形化的Ice Grid定义功能，支持自动调度Ice服务到集群的Node节点上
 - 3.支持Ice服务故障自动恢复的机制，模仿Kubernetes的机制，监测Ice服务的状态，当服务不可用时，重新调度生成。
 - 4.支持Ice服务性能监控和告警功能

欢迎Ice精英和Web开发经验丰富的大侠参与， 将此平台打造成Ice最强管理平台，并被Zeroc 官方认可。
